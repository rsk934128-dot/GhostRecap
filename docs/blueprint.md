# **App Name**: GhostRecap

## Core Features:

- Archive Dashboard: A centralized interface to view and manage recovered notifications and archived messages synchronized from the mobile device.
- Smart Intent Tool: A generative AI feature that uses an LLM to act as a classification tool, reasoning through message context to label archives as 'Urgent', 'Transactional', or 'OTP' automatically.
- Cloud Sync Storage: A secure centralized repository using Firestore to persist archived logs and user-defined automation rules across sessions.
- Response Rule Builder: A drag-and-drop interface for creating complex auto-reply logic based on keywords, sender identity, and received timing.
- Direct Link Hub: Integration tool providing one-click deep links to messaging services like WhatsApp and Telegram directly from message previews.
- History Recovery Timeline: A visual timeline tool that highlights 'deleted message' events, reconstructing conversations from notification snippets.

## Style Guidelines:

- A sophisticated dark scheme designed for confidentiality. Background: Deep Onyx (#111317); Primary: Vibrant Tech Blue (#6691FF) for readability and trust; Accent: Luminous Cyan (#1AC6FF) for status indicators and active states.
- A bold pairing for high-tech utility. Headlines: 'Space Grotesk' (sans-serif) for a scientific, modern edge. Body: 'Inter' (sans-serif) for maximum legibility in high-density notification lists.
- Minimalist, thin-stroke geometric icons reflecting security and digital forensics themes.
- A multi-pane dashboard layout with collapsible sidebars to maximize vertical space for message scrolling.
- Ghostly micro-interactions where new incoming notifications fade in with a subtle blur effect and high-priority alerts utilize sharp, sudden expansion.
